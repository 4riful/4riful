# Ariful Anik

**Security-minded builder working across automation, Linux systems, web tooling, and practical research.**

I like projects that solve real pain: security workflows that save time, Linux scripts that make systems easier to run, recommendation systems that connect people to the right opportunity, and small utilities that are useful when the normal way breaks.

---

## Current Focus

- Web application security, recon automation, and bug bounty research.
- Linux administration, shell tooling, WSL, Raspberry Pi, and self-hosted workflows.
- Full-stack product work with Python, JavaScript, TypeScript, Node.js, React, and MongoDB.
- Recommendation systems for recruitment and matching workflows.
- Network resilience experiments around VPNs, WireGuard, and restricted connectivity.
